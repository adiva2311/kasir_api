# KASIR API

### BASE URL = https://kasir_api.gajahbumi.web.id/api/health-check

#### API - ROUTES

```bash
// PRODUCTS ROUTES
- Get All Product = GET -> "/products"
- Create Product = POST -> "/products",
- Get Product by ID = GET -> "/products/:id"
- Update Product = PUT -> "/products/:id",
- Delete Product = DELETE -> "/products/:id",
- Get Product by CategoryID = GET -> "/products/category/:id"

// CATEGORY ROUTES
- Get All categories = GET -> "/categories"
- Create categories = POST -> "/categories",
- Get categories by ID = GET -> "/categories/:id"
- Update categories = PUT -> "/categories/:id",
- Delete categories = DELETE -> "/categories/:id",
```
